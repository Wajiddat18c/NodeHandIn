Commands

In this table i will explane all of the interesting commands i have learned from this class

Command Explanation
npm init This is used to crate a new node package. Its very smart to use because it will give you a from to fill and after i will create a json file called package.json where all the necessary infomation about your module will be
npm install This command allows you to download packages from npmjs.com or other sides. When you install a package then you will always get a node_modules folder where all of the code is in. It is recommended that you don't push the node_modules folder to git because it containes a lot of storage
npm start This allows you to start a custom made script from the package.json file
